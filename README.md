# 🖊️ PlanBoard

PlanBoard is a real-time collaborative whiteboard application built using the MERN stack and Socket.IO.  
Multiple users can join the same board, draw together, and collaborate instantly from anywhere.

🌍 **Live Demo:** https://planboard-eta.vercel.app/

---

## 🚀 Features

- 🎨 Real-time collaborative drawing
- 👥 Multiple users per board
- 🟢 Live cursor tracking
- 🔐 Role-based access (Admin / Editor / Viewer)
- 🧹 Admin-controlled board clearing
- 💾 Auto-save canvas to MongoDB
- 🔄 Undo / Redo functionality
- 🌐 Fully deployed (Frontend + Backend)

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Socket.IO

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
